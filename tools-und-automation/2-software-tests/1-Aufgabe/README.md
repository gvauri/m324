# Schreibe eine Python-Klasse File, welche folgende statischen Methoden enthält:

    create_file(filename: str)
    write_to_file(filename: str, text: str)
    read_from_file(filename: str) -> str

1. Schreibe für jede Methode einen Unit Test.

2. Führe die Unit Tests mit Hilfe von make durch.

3. Zur Überprüfung des Test Coverages soll make erweitert werden: Die Test Coverage soll ausgegeben werden, das File .coverage soll nach der Überprüfung gelöscht werden.
