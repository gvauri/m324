# Erstelle ein Makefile, welches diese Anforderungen erfüllt:

- Ein Vorgang erstellt das File ip mit der aktuellen IP-Adresse.
- Ein Vorgang erstellt das File host mit dem aktuellen Hostnamen.
- Wenn die Files ip und host vorhanden sind, wird in einem weiteren Vorgang der Inhalt der beiden Files in das File address gespeichert. Dabei wird folgendes, beispielhaftes Format genutzt: lab@192.168.0.10
- Mit dem Vorgang clean werden die Files ip, host und address
gelöscht.
