# 4 Abhängigkeiten

### Welche Probleme können auftreten, wenn beim Ausführen von Softwares die notwendigen Abhängigkeiten fehlen?
Der Test wird Fehlschlagen. Er kennt dann nicht was aufgerfen werden soll.

### Ein Python Code soll in einem Container ausgeführt werden können. Die notwendigen Packages werden automatisch im Container installiert.

    Erstelle einen beliebigen Python Code, welcher mindestens zwei Packages verwendet.

    Erstelle ein Dockerfile, welches die erforderlichen Packages installiert.

    Erstelle ein Makefile, welches das Docker Image erstellt.
